02 — Proyecto personal
personal project

NimbusView

Plataforma meteorológica full-stack en tiempo real. Integra múltiples APIs externas, un backend propio como capa de orquestación y un frontend rico con mapa interactivo, radar animado y temas dinámicos según el clima.

1
Mapa interactivo con Leaflet. Selección por clic, geolocalización, buscador de ciudades y capas meteorológicas con control de opacidad.
2
Backend en ASP.NET Core. Capa de integración con OpenWeather y RainViewer: clima actual, pronóstico a 24h, 5 días y radar animado.
3
Resiliencia de red. Fallback de tiles, cooldown automático ante errores y caché de frames de radar para UX fluida ante fallos externos.
4
UX visual rica. Animaciones, temas dinámicos según condición climática y panel inferior contextual.
React TypeScript ASP.NET Core Leaflet OpenWeather API RainViewer API
